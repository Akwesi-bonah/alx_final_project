ALX Software Engineering Final project (cohort 13)

Hostel Management System

This is the final portfolio pitch of alx software engineering cohort 13. The project is a hostel management system that allows students to book for hostel rooms and also book for vacation stay. The project is built with python flask and mysql database. The project is digital ocean droplet and the source code is available on github.
